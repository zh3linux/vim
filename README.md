Vim
============
My Vim etc

Install
------
Setp 1:

git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle

Setp 2:

git clone https://github.com/zh3linux/vim.git

mv vim/.vimrc ~/.vimrc

Setp 3:

Open Vim and run :BundleInstall

Done.
